# 수와 식 

- 미분은 함수의 극한, 함수를 알아야 미분도 알수 있음, 
    - 함수의 종류는 삼각 함수, 지수/로그 함수가 있음 
- 함수는 **수와 식**을 알아야 함 
- 고등 수학에서의 **수와 식**은
    - 이차식 
    - 인수분해 
    - 곱셈공식 


## 1. 곱셈 공식 

정의 : 간단히 식을 **전개** 하는것, 반대로 하는건 인수 분해 

- 곱셈 공식 :  (x+a)(x+b) -> x^2 + (a+b)x + ab 
- 인수 분해 : x^2 + (a+b)x + ab -> (x+a)(x+b)
