# 수학 기초 

![](https://i.imgur.com/lNGqzcU.png)

- 고등수학의 최종 목표는 미분 적분이다. 
- 미분은 함수의 극한이다. 함수는 y = f(x)로 수와 식으로 이루어져 있다. 
    - 함수의 종류는 삼각 함수, 지수/로그 함수가 있음 
- 함수는 두 집합사이의 대응 관계 **수와 식**을 알아야 함
- 수와 식은에서는 
    - 이차식
    - 인수분해 
    - 곱셈공식 
- 수를 이해 하기 위해서 집합, 수의 체계를 알아야 한다.
